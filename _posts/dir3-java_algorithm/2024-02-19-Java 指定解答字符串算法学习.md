# a 字符串被包含在 b 字符串中
```text
示例 1：

输入：s = "abc", t = "ahbgdc"
输出：true
示例 2：

输入：s = "axc", t = "ahbgdc"
输出：false
```

```java
@Test
public  void test() {
    String s = "abc", t = "ahbgdc";

    int sPoint = 0,tPoint = 0;
    while (sPoint < s.length() && tPoint<t.length()) {
        if (s.charAt(sPoint) == t.charAt(tPoint)){
            sPoint++;
        }
        tPoint++;
    }
    System.out.println(sPoint== s.length());
}
```


# 判断s 是t 的连续子串
 
```java
@Test
public void test() {
    String s = "abc", t = "ahabcgdc";

    int sPoint = 0, tPoint = 0;
    while (tPoint < t.length()) {
        if (s.charAt(sPoint) == t.charAt(tPoint)) {
            sPoint++;
            if (sPoint == s.length()) {
                break;
            }
        } else {
            sPoint = 0;
        }
        tPoint++;
    }
    System.out.println(sPoint == s.length());
}
```
